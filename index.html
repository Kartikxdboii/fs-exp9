How It Works

  • Frontend is built with Node.js and served by Nginx
  • Backend is a Spring Boot JAR built with Maven (multi-stage Dockerfile)
  • MySQL runs as a separate container with persistent volume
  • All 3 containers run on the same Docker network (myapp-network)
  • Run everything with: docker-compose up --build
